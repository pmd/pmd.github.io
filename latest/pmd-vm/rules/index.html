PMD Rulesets index: Current Rulesets

List of rulesets and rules contained in each ruleset.

  • Basic Velocity: The Basic Velocity ruleset contains basic rules for Apache Velocity pages.

Basic Velocity (vm)

  • AvoidDeeplyNestedIfStmts: Avoid creating deeply nested if-then statements since they are harder to read and error-prone to maintain.
  • CollapsibleIfStatements: Sometimes two consecutive ‘if’ statements can be consolidated by separating their conditions with a boolean short-circuit operator.
  • ExcessiveTemplateLength: The template is too long. It should be broken up into smaller pieces.
  • AvoidReassigningParameters: Reassigning values to incoming parameters is not recommended. Use temporary local variables instead.
  • EmptyIfStmt: Empty if statements should be deleted.
  • EmptyForeachStmt: Empty foreach statements should be deleted.
  • UnusedMacroParameter: Avoid unused macro parameters. They should be deleted.
  • NoInlineJavaScript: Avoid inline JavaScript. Import .js files instead.
  • NoInlineStyles: Avoid inline styles. Use css classes instead.
